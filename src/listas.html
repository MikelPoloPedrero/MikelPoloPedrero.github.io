Base de datos Relacionales

  • Es un tipo de sistema de gestión de bases de datos que utiliza un modelo de datos relacional para organizar y almacenar los datos. Este modelo se basa en tablas con filas y columnas, donde cada fila representa un registro y cada columna representa un atributo o campo específico. Los datos se almacenan en tablas relacionadas que pueden estar vinculadas mediante claves primarias y foráneas para establecer relaciones entre ellas.
